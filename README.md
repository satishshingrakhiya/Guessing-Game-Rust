# Guessing-Game-Rust
Rust Starter Project - This is guessing game where you need to guess a number between 1-100 and it gives you hint if its greater than or less than answer untill guess is correct.

## Prerequisite
1. Rust and Cargo installed

## How to run project
1. Clone the repository.
2. Go to project directory and open terminal
3. Run command: Cargo run
4. Enter guesses untill its correct
